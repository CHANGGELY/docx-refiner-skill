## 核心工作流程

整个处理过程分为4个标准化阶段：

### 阶段1：智能提取
```python
python scripts/extract_docx.py input.docx
```
- 提取所有文本内容
- 保存嵌入图片
- 生成结构化JSON索引

### 阶段2：上下文构建
```python
python scripts/build_context.py raw_content.json
```
- 转换为AI可读格式
- 插入图片占位符
- 保持文档顺序

### 阶段3：AI处理
- 按主题聚类内容
- 保留作者风格
- 处理公式和图片
- 生成Markdown初稿

### 阶段4：后处理
```python
python scripts/post_process.py output.md
```
- 验证图片链接
- 添加元数据
- 最终格式化

## 技术亮点

### 1. 零删减原则
所有原始内容都会保留，包括：
- 个人吐槽和观点
- 技术细节和代码
- 图片和公式
- 灵感闪现的片段

### 2. 风格保留算法
通过精心设计的Prompt，AI会：
- 识别并保留作者的语言习惯
- 将口语化表达适当优化
- 维持技术术语的准确性
- 不过度正式化

### 3. 公式双显示
数学公式自动生成两种格式：
- LaTeX：便于技术文档
- Unicode：确保通用显示
