# Product Guide

## 1. 产品定义
**项目名称**：AI 智能文档润色与重组工具 (Docx-to-Markdown Refiner)
**核心目标**：创建一个自动化工作流（Skill），利用多模态大模型能力，将包含杂乱图文的 Word 文档转换为专业的 Markdown 文档。
**用户**：用户本人（通过 CLI 交互）。

## 2. 核心功能
*   **文档解析**：编写 Python 脚本自动提取 `.docx` 中的文本段落和图片资源。
*   **智能 OCR 与润色**：
    *   将提取的内容（特别是图片）发送给大模型进行识别和上下文理解。
    *   **排版还原**：最大程度保留文字在原本图片或文档中的相对位置和逻辑结构。
    *   **专业润色**：在不改变原意、不删减信息的前提下，提升文字的专业度，并将相关的“随笔”归纳整理。
*   **数学公式特殊处理**：
    *   对于图片或文本中的数学公式，输出两种格式：
        1.  **原始 LaTeX**：方便复制 (`$$...$$`)。
        2.  **直观渲染形式**：方便肉眼阅读（如使用 Unicode 符号或简化的文本表示）。

## 3. 约束条件
*   **零删减**：除非是完全无意义的噪点，否则保留所有信息。
*   **即时性**：代码编写后旨在立即处理当前的 `code之路.docx` 文件。
