## 一、 终端开发环境：从“能用”到“极致美化”

想要效率高，终端必须好。不要守着那个破烂的原始 Shell，增强版的工具才是程序员的生产力工具。

### 1.1 核心 CLI 工具链
这些工具是我目前安装并重度使用的，极大地提升了文件查找和目录切换的速度：
- **Oh My Posh**：终端提示符美化，没它我写代码都没心情。
- **ripgrep (rg)**：极速全局搜索，比那个老掉牙的 grep 快出几个量级。
- **fd**：更易用的文件查找，语法比 find 简单多了。
- **bat**：带语法高亮的 `cat`，看代码配置文件的利器。
- **zoxide**：目录“瞬移”，它能记住你常去的目录，直接 `z <目录名>` 就过去了。
- **fzf**：模糊搜索的核心引擎，万物皆可模糊匹配。

### 1.2 PowerShell 增强模块
如果你在 Windows 或跨平台环境下使用 PowerShell，以下模块必装：
- **Terminal-Icons**：为目录添加漂亮的图标，视觉效果拉满。
- **PSFzf**：为 fzf 绑定 PowerShell 键位，让搜索如丝般顺滑。

### 1.3 ZSH：Mac 系统的灵魂
ZSH (Z Shell) 是 Bash 的“升级版”，也是现在 Mac 默认的 Shell 程序。它自带的智能补全（比如 `cd Do` 自动补全 `Documents`）和强大的插件支持（显示 Git 分支、命令历史搜索）是每个程序员高效工作的基石。

> **注意**：不要把“系统终端”和“Python 终端”搞混。当你输入 `python3` 进入交互式解释器后，那是 Python 的地盘，只能执行 Python 代码，别在那瞎输系统命令。

## 二、 量化交易：策略因子与实盘解耦方案

### 2.1 硬核逻辑：多空策略与因子库
多、空、垃（垃圾时间/空仓）是极其硬核的交易逻辑。我的核心因子库包含：
- **VWAP / VWAP Bias / VWAP Bias Std**：成交量加权平均价格及其偏离度。
- **CCI**：在捕捉空头信号时非常管用。
- **CMO**：参考了钱文茂的因子。
- **Supertrend**：Vida 自用的核心因子。
- **MACD**：别听邢不行在那瞎比比说 MACD 没用，他只会无脑选币，完全不懂择时、网格和择时的配合。我在纳指期货上实测 MACD 是管用的。
- **VolumeMeanRatio**：成交量均值比。

> **实操小贴士**：跑下一个策略的时候，记得先把 U 转到待单项目里，这样还能额外白嫖 10% 的分成。

### 2.2 实盘解耦：摆脱 IDE 依赖（方案 B）
如果你受够了“一关 Trae IDE，实盘进程就跟着挂掉”的窘境，请务必执行以下解耦方案：

1. **直接使用系统终端**：打开 Mac 自带的 Terminal 或 iTerm2。
2. **激活环境并启动**：
   ```bash
   cd /Users/chuan/Desktop/xiangmu/客户端
   source Quant_Unified/.venv/bin/activate
   python -X utf8 Quant_Unified/strategies/二号网格策略/real_trading.py
   ```
3. **独立监控日志**：另开一个窗口专门用来挂载日志。
   ```bash
   tail -f logs/grid_strategy_live.log
   ```

**这样做的好处**：IDE 只是你的编辑器，关了、崩了都对实盘没影响。你可以在任何地方随时重新连接这个“跳动”的日志窗口。

## 三、 软技能：字体、会员与模型思考

### 3.1 Mac 字体推荐
在 Mac 上，这些中文字体不仅常见而且排版效果极佳：
- `PingFang SC` (平方)
- `Hiragino Sans GB` (冬青黑体)
- `Songti SC` (宋体)

### 3.2 薅羊毛技巧：GPT 与 GMail 会员
- **GPT**：去搞个 Team 版，一个月也就 10 块钱。
- **GMail**：Business 版也是免费的，或者走家庭组共享学生优惠。
- 基本上 10 块钱以内就能把这两大核心生产力会员搞定，别当大冤种付全价。

### 3.3 大模型深度思考：价格、智商与 Codex 坑
- **思考程度 vs 价格**：同样的模型，Token 价格是一样的。所谓“思考程度”本质上就是让模型输出更多的“思考链（Thoughts）”。
- **Codex 的阉割本质**：`gpt-5.2 codex` 的额度消耗极快，但智商偶尔堪忧，上下文记忆力远不如标准版。每一代 Codex 都感觉是 OpenAI 搞出来的阉割版，慎用。
